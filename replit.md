# Landing Page Project

## Overview
A simple Portuguese-language landing page showcasing services with progress indicators. This is a static HTML site using Bootstrap 5 for styling.

## Project Architecture
- **Type**: Static HTML website
- **Frontend**: Pure HTML/CSS with Bootstrap 5.3.8
- **Server**: Python HTTP server (for serving static files)
- **Port**: 5000

## Files
- `index.html` - Main landing page with service progress bars
- `styles.css` - Custom styling (minimal)
- `sobre.pdf` - PDF file (likely "About" document in Portuguese)
